### Projects

**[NHTSA VIN Decoder V2](https://github.com/ssrpw2/NHTSA-VIN-Decoder-V2)** — Standalone offline VIN decoder using NHTSA's vPIC database. Python + SQLite, no SQL Server required. Decodes make, model, year, electrification level, body class, and 30+ attributes at ~160 VINs/sec. Built for batch research on 100K+ VIN datasets. [V1](https://github.com/ssrpw2/NHTSA-VIN-Decoder) required SQL Server Management Studio, Notepad++, and Excel.

**[On the Calculation of Mechanical Action](https://github.com/ssrpw2/On-the-Calculation-of-Mechanical-Action)** — English translation of Gaspard-Gustave de Coriolis's *Du Calcul de l'effet des machines* (1829), the text that introduced the modern definitions of work and kinetic energy. In progress.

**[Knowledge Graph Explainer](https://isotopyofloops.github.io/minimum-autonomy-stack/index.html)** — Interactive visual explainers for knowledge graph concepts: cosine similarity from first principles, how graph edges form, and how embedding-based retrieval works in practice. Built with real data from a live knowledge graph.

**[P0341 Honda CR-V Timing Fix](https://github.com/ssrpw2/P0341-Honda-CRV-Timing-Fix)** — Complete diagnostic and repair guide for Honda CR-V code P0341, including timing correction, PCM reset, and crank pattern relearn.
